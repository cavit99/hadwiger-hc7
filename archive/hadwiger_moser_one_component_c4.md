# Elimination of the smallest one-component pure-Moser cell

## 1. Statement

### Theorem 1.1 (computer-assisted)

Let \(G\) be a finite simple graph, let \(v\in V(G)\), and suppose

\[
d(v)=7,\qquad G[N(v)]\cong M,
\]

where \(M\) is the pure Moser spindle with edge set

\[
\{01,02,03,04,12,16,26,34,35,45,56\}. \tag{1.1}
\]

Suppose further that

\[
G-N[v]=C
\]

is connected with \(|C|=4\), \(\delta(G)\ge7\), and
\(\kappa(G)\ge7\).  Then \(G\) contains a \(K_7\)-minor.

The proof certificate is generated by `moser_c4_probe.py`, archived in
`moser_c4_certificate.json`, and independently replayed by
`moser_c4_verify.py`.

## 2. Why order four is the first possible order

In the labelling (1.1), deleting

\[
C\cup\{v,0,6\}
\]

separates the edge \(12\) from the triangle on \(\{3,4,5\}\).  Hence
seven-connectivity gives

\[
|C|+3\ge7,
\]

so \(|C|\ge4\).  Theorem 1.1 eliminates this equality case.  Therefore a
one-component pure-Moser cell inside a hypothetical minimal counterexample
must satisfy

\[
\boxed{|C|\ge5.} \tag{2.1}
\]

The subsequent independently certified order-five theorem in
`hadwiger_moser_one_component_c5.md` raises the current residual bound to
\(|C|\ge6\).

## 3. Finite encoding

There are six connected unlabelled graphs on four vertices.  The
certificate treats all six.  For each fixed graph \(C\), the only
undetermined edges are the 28 possible incidences between the seven
vertices of \(N(v)\) and the four vertices of \(C\).  Edges inside
\(N[v]\), inside \(C\), and the absence of edges from \(v\) to \(C\) are
fixed.

The initial Boolean system imposes only necessary conditions:

1. every vertex has degree at least seven; and
2. every boundary vertex has a neighbour in \(C\).

The second condition also follows from seven-connectivity: otherwise the
at most six boundary neighbours of \(C\) separate it from \(v\).

The search then adds two kinds of logically valid clauses.

### Connectivity clauses

If a current graph has a vertex cut \(X\) of order at most six, its
components in \(G-X\) are recorded.  Any seven-connected completion must
add at least one still-optional boundary--\(C\) edge between two of those
components.  The disjunction of all such optional cross-edges is added.

### Minor clauses

If the current graph contains a \(K_7\)-model, seven explicit disjoint
connected branch sets are recorded, together with a set \(W\) of optional
edges sufficient for all branch-set trees and all 21 pairwise
adjacencies.  Every graph retaining all edges of \(W\) contains that same
model.  Consequently a \(K_7\)-minor-free graph must omit at least one edge
of \(W\), and the clause

\[
\bigvee_{e\in W}\neg e
\]

is added.

Each clause is therefore necessary for a seven-connected,
\(K_7\)-minor-free completion.  After finitely many clauses the Boolean
system is unsatisfiable in every one of the six cases.

## 4. Certificate totals and independent audit

In increasing graph-atlas order for the six connected four-vertex graph
types, the numbers of recorded clauses are

\[
37,\quad40,\quad88,\quad113,\quad154,\quad180. \tag{4.1}
\]

The independent verifier checks all of the following without invoking the
model-search routine:

* the archive contains exactly the six connected unlabelled graph types;
* every connectivity record is a partition after deleting at most six
  vertices, has no fixed cross-edge, and lists every possible optional
  cross-edge;
* every minor record consists of seven disjoint connected bags which are
  pairwise adjacent using only fixed edges and the recorded optional
  edges; and
* after adding the verified clauses to the degree/attachment system, Z3
  returns `unsat` in each case.

Running

```text
PYTHONPATH=.deps python3 moser_c4_verify.py
```

prints

```text
verified 0 37 clauses
verified 1 40 clauses
verified 2 88 clauses
verified 3 113 clauses
verified 4 154 clauses
verified 5 180 clauses
```

The theorem uses no colouring-extension assumption beyond the audited
minimal-counterexample consequences \(\delta(G),\kappa(G)\ge7\).  It is a
complete elimination of the smallest one-component pure-Moser exterior,
not a proof for \(|C|\ge5\).
