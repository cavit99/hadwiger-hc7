# Supporting goal: shortest-lock state-preserving first-seam receiver

**Status:** paused after the tight-lobe transition. The active semantic
target is now the
[exact-seven state-selection theorem](hc7_exact7_state_selection_goal.md).

## Decision

The literature review through 15 July 2026 found no theorem that preserves
both literal rooted geometry and an exact six-colour boundary state.  The
immediate target is therefore the first unsupported semantic step left by
the audited compulsory-lock programme: in one fixed colouring, decode the
first genuine crossing or first crossless web seam of a shortest thin lock
and one second lock into a **recognized receiver**.  Do not compare
independently attained final-duty states.

The first mandatory milestone was the **tight-gate receiver**. A shortest
thin lock and a second lock now give, whenever the second first enters a
tight off-spine lobe, a smaller actual seven-cell with a named one-sided
contracted state; a separating entry couples that state to the fixed
colouring by one explicit Kempe swap. The subsequent packet analysis
regenerates a high-demand state on the actual packet-one shore and gives a
five-colour lock/crossing localization, but independent audit rejects the
claimed accepted `S3` and ranked `S4` normalizations. This certificate is
not yet a descent.

Only after state selection resolves that semantic gap should this
mechanism handle a
boundary-excess lobe, a same-lock bypass, or a spine-confined/rich
excursion by a crossing/web decomposition.  The all-rich articulation
problem remains secondary.

This is the constructive engine for the connected-rich paired width-two
`(1,2)` cell.  The frozen direct-reserve web and further boundary censuses
remain secondary falsifiers only.

## Audited entry

Use the frozen separation

\[
 V(G)=A\mathbin{\dot\cup}S\mathbin{\dot\cup}R,
 \qquad S=W\mathbin{\dot\cup}\{u\},\qquad |S|=7,
\]

where `G` is a hypothetical seven-connected, strongly
seven-contraction-critical, `K_7`-minor-free graph, there is no `A-R` edge,
the rich shore contains two disjoint adjacent `S`-full packets, and the
thin shore contains the unique compulsory edge

\[
                         e=zu,
\]

with `z in A`, `u in S`, and `A-z` connected and `W`-full.  The paired
width-two boundary normalization is in force.

Fix one proper six-colouring `phi` of `G-e`, and put
`phi(z)=phi(u)=alpha`.  For each other colour `delta`, let `K_delta` be
the compulsory `alpha-delta` component containing `z,u`.

The rich separating-bridge branch is already normalized: if some
`K_delta` has a rich-internal or boundary--rich `z-u`-separating bridge,
the audited lock-state lift and root substitution give an actual labelled
`(1,1)` cell with a named attained state and packet-sum drop `3 -> 2`.
The present target therefore assumes that no such bridge exists.

For each `delta`, the audited rich-edge quotient gives exactly one of:

1. a literal `z-u` path in `K_delta-R`, hence a boundary neighbour
   `w_delta in W` of colour `delta`; or
2. two edge-disjoint source--target routes in the rich-edge quotient.

Retained rich edges belonging to distinct colours are disjoint.  The
routes need not be internally vertex-disjoint and may share arbitrary
`alpha`-coloured articulation vertices.  This is the exact starting
certificate, not a rooted model.

## Target theorem

### State-preserving first-seam receiver

Under the audited entry, choose a shortest thin lock `P` and one second
lock `Q`, retaining their complete literal bridge hull.  At the first
off-spine lobe or the first useful parallel-composition seam, at least one
of the following occurs.

1. **Exact reflection.**  Literal contractions or Kempe switches produce
   one equality partition belonging to both closed-shore extension
   languages, and the two colourings glue to a six-colouring of `G`.
2. **Literal terminal.**  There is a literal `K_7` model, or a fixed pair
   `{p,q}` for which `G-{p,q}` is `K_5`-minor-free.
3. **Recognized exact-seven receiver.**  There is an actual
   seven-separation with an explicit old-to-new boundary map and a named
   proper-minor operation, and it is one of:

   * an oriented `(1,2)` cell with a smaller active shore and an attained
     accepted `S3` state;
   * a normalized ranked `(1,1)` `S4` handoff; or
   * the already closed `(1,3)` branch.
4. **Ranked near-model receiver.**  There is a normalized labelled `S1`
   or `S4` handoff together with a rank which strictly decreases under the
   receiving transition.  An unlabelled `K_7^vee`, a naked `(1,1)` cell,
   or a reversible root exchange is not an outcome.

For the present goal the theorem need only cover every branch reached from
one shortest thin lock and one second lock.  The all-rich five-lock branch
is a later obligation.  A smaller cut with only a one-sided contracted
state is not outcome 3.

## Current transition certificate

The audited
[tight bypass-lobe transition](../results/hc7_shortest_thin_lock_tight_bypass_transition.md)
proves the following local core.

For `D` a component off the shortest thin spine, put

\[
 Z_D=N_A(D),\qquad T_D=N_S(D).
\]

Seven-connectivity gives `|Z_D|+|T_D|>=7`.  At equality,
`Omega_D=Z_D union T_D` is a literal actual seven-boundary.  Contracting
any `D-Omega_D` edge toward its boundary endpoint supplies one intact
opposite-side state and the same named contracted lobe-side state, with
packet demand strictly greater than the opposite packet capacity.  If a
second lock's first entry edge separates its ends, one Kempe swap produces
that same state from the fixed colouring.

This is a strict set-size transition only.  The old rich packets need not
be `Omega_D`-full, the lobe need not be the packet-one shore, and the state
need not be the paired-triangle state.  Those three facts are exactly the
receiver obligation.

## Constructive mechanism

### M1. Tight-gate receiver

First close the certificate above.  Determine the actual packet vector of
the new cell and use the separating-entry toggle, the demand inequality,
and model regeneration to obtain one of the recognized receivers in
outcome 3.  A proof may not infer that old `S`-full packets are
`Omega_D`-full, or that `D` is the packet-one shore.  Those facts must be
proved or bypassed.

### M2. Thin-lock-anchored mixed decoder

Choose a shortest thin `z-u` lock path `P_delta` and retain its complete
literal bridge hull in

\[
                 H=(G-zu)[A\cup S];
\]

no chord or off-path component may be discarded.  Couple it to one
different colour `epsilon`:

* if `epsilon` is thin, use the genuine `P_delta`-ears obtained from
  components of the second path outside `P_delta`; or
* if `epsilon` is rich, first prove an injective literal lift/terminal
  lemma, because the two quotient routes may share contracted thin
  components and repeated-colour vertices.

Apply the generalized two-path theorem only to the resulting fully
specified terminal tuple.  A crossing must be decoded into disjoint
duty-faithful carriers or literal rooted bags.  A crossless outcome must
retain the actual web frame and its recursive parallel-composition seams;
completion edges are never host edges.  At the first equality gate, a
named deletion or contraction response must attain the receiving state.

A thin lock supplies a literal rainbow neighbour `w_delta` of `u`; five
thin locks force the connected bipartite boundary into the exact `2+5`
orientation recorded by the quotient theorem, with six equality blocks and
one repeated independent pair.  This is the terminal base pattern for M2,
not by itself a reflection theorem.

### M3. Repeated-colour articulation dispersion

The static all-rich implication is false.  Five edge-disjoint colour
systems can all pass through one `alpha` hub, and off-colour bypass edges
can make that hub nonseparating without changing any two-colour quotient.
Therefore choose an inclusion-minimal **full-host** union of the five locks
and their bypasses; quotient route data alone is not an input to a positive
adhesion theorem.

In that minimal all-rich host, organize each quotient by its block-cut tree.
Intersections between different colour systems can occur only at
`alpha`-coloured vertices or contracted thin components.  Prove that a
first common articulation either:

* disperses, by literal endpoint-preserving augmentation, into the
  vertex-disjoint labelled carriers/rooted bags needed by outcomes 1--2;
  or
* canonically exposes a full six-terminal crossless web seam whose lifted
  gate is an actual separator.

Global seven-connectivity cannot be substituted for the relative
separation hypothesis of an augmentation lemma.  That hypothesis must be
verified in the literal host or its failure must itself supply the gate.

### M4. Stateful web pullback

At the first useful parallel-composition seam, use one named deletion or
contraction response to attain the new seven-boundary state.  The boundary
pairing may be whatever the legal operation returns; it may not be imposed
from the drawing.  The seam succeeds only if it reflects, gives a literal
terminal, or strictly decreases

\[
 (\text{active-shore order},
   \text{unresolved web nodes},
   \text{receiver rank}).
\]

Any transition cycle with this signature unchanged falsifies the proposed
induction.

## Literature boundary

The fresh review through July 2026 found no theorem which performs this
decoder automatically.

* [Humeau--Pous](https://arxiv.org/abs/2505.16431) proves the generalized
  crossing/web dichotomy for an arbitrary terminal tuple and recursively
  generates crossless webs by parallel composition.  It supplies the
  structural engine, not labels or equality-state transfer.
* [Hayashi--Kawarabayashi--Yoo](https://doi.org/10.1137/23M157082X)
  restates Perfect's endpoint-preserving path augmentation.  Its relative
  separation hypotheses must be checked inside the lock geometry.
* [Pap](https://doi.org/10.1007/s00493-007-0056-z) gives a min--max theorem
  for genuinely permutation-labelled non-returning paths.  It is a bounded
  encoding experiment only until every move is total and reversible and
  auxiliary vertex-disjointness lifts injectively to literal vertices.
* [Kawarabayashi--Pedersen--Toft](https://arxiv.org/abs/0810.3133) gives
  prescribed-colour paths in the conditional double-critical branch.  It
  does not supply disjointness or the general `chi(G-{z,u})=6` case.
* [Kriesell--Mohr](https://arxiv.org/abs/1911.09998) warns that pairwise
  two-colour connectivity alone does not package rooted clique models in
  the higher-colour regime.
* Rooted-folio irrelevant-vertex results preserve bounded annotated minor
  geometry, not exact colour-extension languages.  They remain a
  proof-producing certification backup after the state language is fixed.

## Falsification and success tests

Before proving a sublemma, test its static form against the common
`alpha`-hub architecture and the order-four/order-five exact contact
models.  The hub can be given arbitrary off-colour bypasses without
altering any of the five lock quotients, so those quotients cannot by
themselves force an actual separator or attained state.  Such a
counterarchitecture retires only the static form; a positive lemma must
spend full-host minimality, universal proper-minor responses or literal
seven-connectivity.

No more boundary or shore-order enumeration is part of this goal.  A
computer search is useful only to falsify a proposed transition invariant
or to emit independently checkable certificates for a bounded state
system.

Success requires a complete proof and independent audit of the target
theorem, including every literal path lift, branch-set adjacency, boundary
map, state-attainment operation and rank decrease.  Even then `S1`, the
general `(1,1)` receiver and larger adhesions remain separate obligations;
the result would close one genuine infinite `HC_7` family, not by itself
prove `HC_7`.
